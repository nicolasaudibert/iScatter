iScatter - Interactive scatterplots

Nicolas Audibert

Université Sorbonne Nouvelle, département ILPGA

Laboratoire de Phonétique et Phonologie (CNRS & Sorbonne Nouvelle)

Load a data file and select two quantitative variables to display a scatterplot that you can parameterize and explore interactively.
Rectangular selection = display details of the points included in the selection.
Click on the figure = select the nearest point(s) (standardized scales: to ensure that the selected point is always the closest visually, size the window to obtain a square figure).

Note on data protection: Uploaded data is processed on a server hosted in France and administered by the LPP, is not retained after the application is closed or your session expires, and is not processed in any way other than by you using the application. Data transmission between your computer and the server is secured by the HTTPS protocol. However, if you are processing sensitive data, e.g. data including personal information or information that could enable people to be identified, we strongly advise you to use the local version of the application downloaded from GitHub and installed on your computer.
